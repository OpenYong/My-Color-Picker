## 개요 

### 요약

- React 사용

### 기능

- 색상 코드 복사
- 색상 팔레트 생성(Drag and Drop)
- 로컬 스토리지 저장

### 추가해볼 기능

- DB 연동**
- 모바일*
- Tag 혹은 Category 추가


### 개발 도구

* React
    - react-copy-to-clipboard
    - Chroma
    - rc slider
    - Material UI
    - react-colorful
    - @dnd-kit/sortable
    - emoji-mart
- MUI
